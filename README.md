- Add to cart & place orders
- REST API backend
- Responsive UI

## 🚀 Getting Started
```bash
# Clone the repo
git clone https://github.com/sujitmanw/Food-Delivery.git

# Install backend
cd backend && npm install

# Install frontend
cd ../frontend && npm install

# Run both
npm run dev
```
